# Forsetti Apple Overlay

The Forsetti Apple overlay records Apple-platform alignment guidance for Forsetti Agentic Edition.

## Purpose

Apple platform work should preserve the Forsetti source-truth principles while using native Apple implementation patterns where appropriate.

## Source-Truth Principles

Apple-aligned work must preserve:

- native-first execution
- contract-first delivery
- boundary-first design
- policy-first enforcement
- host-agnostic modules
- capability enforcement
- module identity validation
- module-scoped context
- one active UI or app surface unless another model is explicitly documented

## Boundary

This overlay may document Apple-specific execution expectations. It must not redefine portable governance rules, root constitutional authority, task contract requirements, or compliance outcomes.

## Phase 01 Status

This overlay is a scaffold. Detailed platform validation and implementation guidance are reserved for a later remediation phase.
