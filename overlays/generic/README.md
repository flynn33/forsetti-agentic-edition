# Generic Overlay

The generic overlay is the default host-neutral operating profile for Forsetti Agentic Edition.

## Purpose

Use this overlay when no platform-specific overlay is selected. It keeps guidance limited to requirements that can be satisfied on ordinary local development environments and continuous integration systems.

## Operating Expectations

- Use local repository files as canonical source.
- Use task contracts for meaningful work.
- Keep changed files inside contract scope.
- Produce validation evidence before claiming completion.
- Record known issues and limitations.
- Keep host-specific setup outside the portable core.

## Tooling Position

The generic overlay does not require a specific shell, IDE, browser, MCP server, container runtime, or hosted workflow provider. Teams may use local tools that produce verifiable evidence.

## Phase 01 Status

This overlay is a scaffold. It documents default boundaries only.
