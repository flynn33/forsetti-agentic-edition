# Forsetti Agentic Portable Core Constitution

## Authority Boundary

This document is a portable-core scaffold. It is subordinate to root `FORSETTI_CONSTITUTION.md` and does not amend, replace, supersede, or reinterpret the root constitution.

If this document conflicts with root `FORSETTI_CONSTITUTION.md`, the root constitution controls.

## Purpose

The portable core constitution records the governance invariants that adapters and overlays must preserve when Forsetti Agentic Edition is applied outside a single hosting model.

## Core Invariants

### Contract Before Action

Meaningful work begins under a task contract that defines scope, authority, required outputs, and validation evidence.

### Scope Is Binding

The task contract defines the authorized work boundary. Files outside that scope must not be modified unless the contract is amended before execution.

### Evidence Determines Compliance

Completion, validation, documentation status, and release impact must be supported by observable evidence.

### Roles Have Boundaries

Planning, building, validation, release, documentation, and stewardship responsibilities must remain distinguishable even when one contributor performs multiple duties.

### Documentation Is Delivery

Changes that affect repository understanding, governance usage, release history, or integration structure require documentation updates in the same governed work unit.

### Adapters Are Optional

Adapters translate host-specific context into portable governance inputs. They do not define canonical policy meaning.

### Overlays Narrow Execution

Overlays document platform-specific execution expectations while preserving core governance meaning.

## Dependency Rule

The portable core must not depend on an adapter, overlay, hosted workflow runner, IDE, MCP server, browser, container runtime, or provider-specific tool. Those tools may be used to produce evidence in a governed task, but they are not core product dependencies.

## Scaffold Status

This Phase 01 document establishes boundaries only. It does not introduce a new constitutional amendment process, change role authority, or implement validation behavior.
